# 🏏 IPL Exploratory Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Indian Premier League (IPL) datasets using Python. The goal is to extract meaningful insights about teams, players, match outcomes, and overall tournament trends through data visualization and statistical analysis.

---

## 🎯 Objectives

- Analyze IPL match and player data
- Explore team performances across seasons
- Identify top run scorers and wicket takers
- Study toss decisions and their impact on match outcomes
- Perform feature engineering for bowling statistics
- Visualize important trends using charts

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed

### Dataset Exploration
- Inspected dataset structure
- Checked missing values
- Generated descriptive statistics

### Team Analysis
- Matches won by each team
- Toss winners analysis
- Toss decision distribution
- Home vs Away performance

### Player Analysis
- Top run scorers
- Top wicket takers
- Best bowling figures
- Player performance comparison

### Visualizations
- Bar Charts
- Count Plots
- Histograms
- Boxplots
- Pie Charts
- Scatter Plots

### Feature Engineering
- Extracted wickets from bowling figures
- Created new analytical features for better insights

## 📈 Key Insights

- Identified the most successful IPL teams.
